Setup Required

  1. Open (or create) your Google Sheet with columns:
    name, big, lin, pledge_class, class_year
  2. Go to File → Share → Publish to web
  3. Choose your sheet tab, set format to Comma-separated values (.csv), click Publish
  4. Copy the published URL
  5. Open docs/js/config.js, find SHEET_URL near the top, and replace the placeholder with your URL
  6. Push to GitHub and enable Pages under Settings → Pages, source: docs/ folder, main branch

The sheet is fetched live on every page load — no rebuild needed when data changes.
Make sure the sheet is shared as "Anyone with the link can view".

Could not load data

  • Confirm the Google Sheet is published (File → Share → Publish to web)
  • Confirm the sheet is shared as "Anyone with the link can view"
  • Check the SHEET_URL value in docs/js/config.js
